# PyPaint

PythonとPySide6を使用した、画像処理エフェクト機能付きのペイントアプリケーションです。 通常の描画機能に加え、画像の回転やぼかしをリアルタイムで適用しながら、その上から正確に描画を続けることが可能です。

**Developed in:** February 2026 (-week project)
## 🚀 主な機能
ペイント機能

自由線描画（ブラシ）

Shiftキー + ドラッグ による直線描画

消しゴムモード（現在の背景色に適応）

ブラシサイズ・色のカスタマイズ

画像処理エフェクト(ぼかし)

回転（-180° 〜 180°)

背景の色の変更

画像を回転させた状態でも、ペイントできるように

既存画像の読み込み（PNG/JPG/JPEG）

保存機能

やり直し機能

## 🛠 使用技術
Python

PySide6

OpenCV

NumPy

## 📦 セットアップ・実行方法
ライブラリのインストール
```bash
pip install PySide6 opencv-python numpy
```
アプリケーションの起動
```bash
python main.py
```
💡 使い方
描画: キャンバス上を左クリックでドラッグします。

直線: Shift キーを押しながらドラッグすることで、始点から終点まで直線を引けます。

背景変更: 「背景色を変える」ボタンからキャンバス全体の色を変更できます（消しゴムの色も自動追従します）。

エフェクト: 右側のスライダーで「回転」や「ぼかし」を調整します。回転したまま描き込むことも可能です。

設計・実装に要した作業時間: 20時間
